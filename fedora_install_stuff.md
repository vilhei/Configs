# Fedora guide

## Install fedora
- Use Fedora media writer to write iso to usb drive.
- Install fedora
- Remember to choose correct keyboard layout

## Install zsh
```shell
sudo dnf install zsh
```

## Install oh-my-zsh
```shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
